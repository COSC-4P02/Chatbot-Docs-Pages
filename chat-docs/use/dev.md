# Start Development Server

## Backend

<code-group>
<code-block title="NPM">
```bash
cd Backend
pip3 install -r requirement.txt
npm install
npm run serve
```
</code-block>
</code-group>

## Frontend

<code-group>
<code-block title="YARN">
```bash
cd Frontend
yarn
yarn serve
```
</code-block>
</code-group>
