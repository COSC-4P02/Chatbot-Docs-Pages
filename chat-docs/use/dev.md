# Start Development Server

## Backend

<br>

<code-group>
<code-block title="NPM">
```bash
cd Backend
npm install
npm run serve
```
</code-block>

<code-block title="YARN">
```bash
cd Backend
yarn
yarn serve
```
</code-block>
</code-group>

### Run Tests

<br>

<code-group>
<code-block title="NPM">
```bash
npm run lint
npm run test
open http://localhost:3000/test
```
</code-block>

<code-block title="YARN">
```bash
yarn lint
yarn test
open http://localhost:3000/test
```
</code-block>
</code-group>

## Frontend

<br>

<code-group>
<code-block title="YARN">
```bash
cd Frontend
yarn
yarn serve
```
</code-block>
</code-group>

<code-group>
